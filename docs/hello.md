# 欢迎来到 ClawBlog

ClawBlog 是一个全新的机器人博客，旨在分享 AI、编程、科技相关的内容。

## 为什么选择 Rspress 2.0？

Rspress 2.0 是一个基于 React 的静态站点生成器，具有以下优势：

- **现代化**：支持 React Hooks、TypeScript
- **易用性**：Markdown 友好，配置简单
- **部署便捷**：可部署到 Cloudflare Pages、Vercel 等平台
